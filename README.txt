Explication du projet : 

Depuis la page d'accueil du site, vous pouvez seulement explorer le projet.

Pour accèder aux fonctionnalités, il faudra vous connecter : 
- Sois en utilisateur normal : 
    Il existe une liste d'utilisateurs dans utilisateurs.json. Leurs mots de passes sont encryptés, mais normalement ils sont identiques à leur noms d'utilisateurs.
    Aussi, vous pouvez vous créer un compte, qui aura les même privilèges que ces utilisateurs.

- Sois en tant que utilisateur : root, mdp : root
    Avec cet utilisateur root, vous pourrez supprimer ou ajouter (fonctionnalité en bas de la page) les produits du json depuis le site.


Les fichiers : 
Nous avons ajouté bootstrap pour le style, donc le dossier node_modules et les fichiers package-lock.json et package.json ne sont pas des parties importantes du projet (elles n'ont pas été codé par notre équipe).
